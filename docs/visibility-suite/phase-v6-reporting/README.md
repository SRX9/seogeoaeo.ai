## Phase exit criteria

- User can generate an in-app + PDF report, see month-over-month deltas, run a competitor
  comparison, and one-click-improve a weak passage with a **verified citability lift**.
- The proof stack renders end-to-end: score delta (V6.3) + answer share (V5.5) + real
  traffic with audit-date markers (V6.6) on `/visibility` and in the digest/reports.
