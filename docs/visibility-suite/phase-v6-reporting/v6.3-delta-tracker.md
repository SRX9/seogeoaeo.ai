# V6.3 — Monthly delta / progress tracker

- **Status:** ☐ Not started
- **Type:** ⚙️ Deterministic
- **Pillar:** ⚪ Cross-cutting · Catalog #31

- **Unlocks:** recurring-value/subscription justification, V7.3

## Goal

Compare a baseline audit to the current one; show deltas with trend arrows across every
category/platform, action-item completion, and a 6-month trajectory. **This proves the gain.**

## Files to create

- `src/lib/visibility/compare.ts` — `compareAudits(baselineId, currentId): DeltaReport`.
- a dashboard "Progress" view + Markdown export.

## Build steps

1. Load two versioned `audits` rows (oldest = baseline, newest = current; if only one exists, run a
   fresh quick audit as current).
2. Compute deltas for overall + all six category scores + all five platform scores + crawler status.
3. Assign **trend symbols:** ▲▲ strong improvement · ▲ improvement · ── unchanged · ▼ decline ·
   ▼▼ significant decline.
4. Track action-item completion (resolved findings) across quick wins / medium / strategic.
5. Include a 6-month trajectory + a conservative business-impact estimate (citation likelihood,
   crawler coverage, estimated traffic value).
6. Test: deltas + trend symbols correct on two fixture audits.
