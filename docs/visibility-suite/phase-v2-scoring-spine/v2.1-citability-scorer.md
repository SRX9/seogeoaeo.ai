## Goal

Split a page into heading-bounded blocks, score each **0–100** across five weighted dimensions,
surface top/bottom blocks + page-level metrics, and (LLM) generate rewrite suggestions for weak
blocks. **The deterministic score must be reproducible** — same HTML → same number, forever.

## Files to create

- `src/lib/visibility/citability.ts` — `scorePassage(text, heading?)` + `analyzePageCitability(html)`.
- `src/lib/visibility/citability.test.ts` — **lock the algorithm** with fixed fixtures + exact scores.
- `src/lib/visibility/citability-rewrite.ts` — LLM `light` rewrite suggestions (separate; never
  touches the deterministic score).

## Build steps — port `score_passage()` 1:1

Implement each dimension with its **cap** and **exact regexes/thresholds**, then sum (max 100):

1. **Answer Block Quality (cap 30):** +15 if a definition pattern matches (`X is a/an/the`,
   `refers to`, `means`, `defined as`, `in simple/other terms,`); +15 if answer appears in first 60
   words (verb/`%`/`$`/`million|billion|thousand`); +10 if heading ends with `?`; + `clarity_ratio*10`
   (share of sentences 5–25 words); +10 if attribution (`according to|research shows|studies show…`).
2. **Self-Containment (cap 25):** word-count band — **134–167 → 10**, 100–200 → 7, 80–250 → 4,
   `<30`or`>400` → 0, else 2; pronoun ratio `<0.02 → 8`, `<0.04 → 5`, `<0.06 → 3`; proper-noun count
   `≥3 → 7`, `≥1 → 4`.
3. **Structural Readability (cap 20):** avg sentence length 10–20 → 8 / 8–25 → 5 / else 2;
   +4 transition words; +4 numbered/step markers; +4 if paragraph breaks (`\n`).
4. **Statistical Density (cap 15):** `%` count ×3 (max 6); `$` amounts ×3 (max 5); numbers-with-unit
   ×2 (max 4); year `20(1\d|2[3-6])` → +2; named sources (`according to|Gartner|McKinsey|…|(Author 2024)`) → +2.
5. **Uniqueness (cap 10):** +5 original-research (`our study/data…`, `we found/analyzed…`);
   +3 case-study/example; +2 specific tool/product (`using/with/via [A-Z]…`).
6. **Grades:** ≥80 A "Highly Citable" · ≥65 B · ≥50 C · ≥35 D · else F.
7. **Page aggregation (`analyzePageCitability`):** split via V0.2 `extractContentBlocks`; score each;
   page score = **average of top-5 blocks** (or all if `<5`); also return grade distribution +
   count of optimal-length (134–167) passages + top-5/bottom-5.
8. **Rewrite (separate, LLM `light`):** for blocks `<60`, generate a suggested opening sentence +
   concrete fixes. Keep deterministic score and LLM output in different functions/fields.
