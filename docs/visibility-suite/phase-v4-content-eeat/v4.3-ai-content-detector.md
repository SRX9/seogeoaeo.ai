## Goal

Flag low-effort AI patterns and rate human-vs-AI likelihood — our own quality gate before we
publish AI-written content. **Never make a definitive "this is AI" claim.**

## Files to create

- add `detectAiContent(snapshot): AiContentResult` to `src/lib/visibility/content.ts`.
- Zod schema + fixture test.

## Build steps

1. `light`-tier classifier prompt that checks the **red-flag table** and returns
   `{ redFlags:[{indicator, evidence?}], label }` where label ∈ {Highly Likely Human,
   Likely Human-Edited AI, Likely AI with Light Editing, Likely Unedited AI}.
2. Red flags: generic phrasing ("in today's digital landscape", "delve into"), lack of specifics,
   no original data, perfect-structure/empty-substance, hedging overload, no authorial voice,
   repetitive thesis, keyword stuffing.
3. Output the label + evidence; **frame as likelihood, never certainty** (per the agent's caveat).
4. Test: a generic fixture → multiple red flags + "Likely Unedited AI"; a rich first-person piece →
   "Highly Likely Human".
