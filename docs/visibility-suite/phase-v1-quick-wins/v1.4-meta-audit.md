# V1.4 — Meta tags & Open Graph auditor

- **Status:** ☐ Not started
- **Type:** ⚙️ Deterministic
- **Pillar:** 🔵 SEO (also 🟣 AEO) · Catalog #8
- **Depends on:** [V0.1](../phase-v0-foundation/v0.1-page-fetcher.md)
- **Unlocks:** V1.5, V2.2 (meta sub-score)

## Goal

Audit title, description, canonical, viewport, `lang`, hreflang, robots directives, Open Graph and
Twitter Card — checking **presence and quality** (a title of "Home"/"Untitled" = effectively missing).

## Files to create

- `src/lib/visibility/meta-audit.ts` — `auditMeta(snapshot): MetaAuditResult`.
- `src/lib/visibility/meta-audit.test.ts`.

## Build steps

1. Read meta from the V0.1 `PageSnapshot` (no fetch).
2. Check each tag against the rule table (below): title present + 50–60 chars + not a placeholder;
   description present + 150–160 chars; canonical self-referencing/preferred; viewport
   `width=device-width, initial-scale=1`; `html lang`; robots directives (flag `noindex`); OG set
   (title/description/image/url/type); Twitter Card set; hreflang if multilingual.
3. Emit per-tag status (Present/Missing/Issue) + findings with severity (missing title/noindex =
   high) + a `fix_payload` of suggested tag values where derivable.
4. Tests: missing title, over-length description, `noindex`, placeholder title → correct flags.

## 📐 Build from (source of truth)

`inspiration-code/agents/geo-technical.md` → **Step 3** (the full meta-tag check table with target
lengths and "issue if missing/wrong" column) and the **"Important Notes"** (presence vs quality —
`"Home"`/`"Untitled"` is effectively missing; `noindex` excludes the page).

## Acceptance
