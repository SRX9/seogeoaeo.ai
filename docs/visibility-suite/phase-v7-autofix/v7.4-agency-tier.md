## Goal

White-label reports, a prospect CRM, and proposal generation for agency/reseller users.
**Per the north star this is an optional premium add-on, not core — build only after V0–V6 land.**

## Files to create

- `src/lib/visibility/brand-config.ts` — workspace white-label config (name, contact, colors, logo).
- CRM tables + dashboard (prospects, pipeline, MRR).
- `src/lib/visibility/proposal.ts` — proposal generator from audit data.

## Build steps

1. **White-label (#30):** a workspace brand config deep-merged over defaults; feed into V6.2 PDF +
   V6.1 report.
2. **CRM (#32):** pipeline stages (lead → qualified → proposal → won/lost), GEO score, notes, MRR;
   a dashboard with filter/sort + PDF download. `/geo quick` (V1.5) writes the prospect score.
3. **Proposal (#33):** generate a tiered proposal from audit data (findings, packages with pricing,
   ROI projection, timeline); recommend a tier from the score.
4. Gate the whole tier behind a premium plan (Stripe, content-roadmap Phase 1).
