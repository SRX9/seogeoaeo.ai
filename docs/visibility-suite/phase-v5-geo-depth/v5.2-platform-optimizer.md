## Build steps

1. One `heavy` evaluation per platform (or one structured multi-platform call) returning sub-scored
   JSON. Feed in the snapshot + brand signals (V5.1) + crawler access (V1.1) so the model doesn't
   re-derive them.
2. **Port each platform's sub-score breakdown exactly:**
   - **Google AIO:** Content Structure 40 (Q-headings, 40–60 word answer paragraphs, tables) /
     Source Authority 30 / Technical 30.
   - **ChatGPT:** Entity Recognition 35 (Wikipedia/Wikidata/sameAs) / Content 40 / Crawler Access 25.
   - **Perplexity:** Community Validation 30 (Reddit/Quora/SO) / Source Directness 30 / Freshness 20 /
     Technical Access 20.
   - **Gemini:** Google Ecosystem 35 (YouTube/GBP/Scholar) / Knowledge Graph 30 / Content 35.
   - **Bing Copilot:** Bing Index 30 (IndexNow, `msvalidate.01`) / Content 30 / MS Ecosystem 20 /
     Technical 20.
3. Compute the Platform Readiness Average; identify strongest/weakest + **cross-platform synergies**
   (e.g., Wikipedia helps ChatGPT + Perplexity + Gemini) + per-platform quick wins.
4. Persist `platform_scores` rows. Test: a fixture with a Wikipedia entity boosts ChatGPT/Gemini.
