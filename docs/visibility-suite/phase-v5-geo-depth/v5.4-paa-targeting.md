## Goal

Mine the real questions users ask for a topic and map each to a **heading + answer block** the page
should contain — so every article ships with a built-in AEO outline.

## Files to create

- `src/lib/visibility/paa.ts` — `buildQuestionOutline(topic): QuestionOutline`.
- test.

## Build steps

1. Pull real questions from the existing research provider (content-roadmap Phase 4: PAA,
   autocomplete, forum questions).
2. LLM `light`: cluster + dedupe questions; for each, produce a question-style H2/H3 + a target
   **40–60 word direct answer** spec (the answer-target pattern).
3. Score each proposed answer block with V2.1 citability so we draft toward high-citability shapes.
4. Output an outline the writer (content-roadmap Phase 3) consumes; feed gaps to the backlog.
5. Test: a topic yields a deduped question outline with answer-target specs.
